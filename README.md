A smart web app built using Flask and NLP that analyzes resumes, calculates match scores with job descriptions, and recommends relevant job openings.

## Features
- Resume text extraction (PDF/DOCX)
- Match score calculation using cosine similarity
- Job recommendations based on resume content
- User authentication (login/register)
- Admin panel for job management

## Tech Stack
- Python (Flask)
- NLP: spaCy, scikit-learn
- Database: SQLite
- Frontend: HTML, CSS, Bootstrap

## How to Run
1. Clone this repository  
   `git clone https://github.com/<your-username>/resume-analyzer.git`
2. Navigate to folder  
   `cd resume-analyzer`
3. Install dependencies  
   `pip install -r requirements.txt`
4. Run app  
   `python app.py`

Access at: **http://localhost:5000**
